## Overview

This application displays historical temperature data from the Central England Temperature (CET record, the longest instrumental temperature series in the world dating back to 1659. Explore temperature trends through different views and interactive features.

## Views

1. **Monthly Table** — View monthly temperature data organized by decade, with color-coded temperatures and annual averages.
2. **Temperature Chart** — Visual bar charts showing recent annual temperatures and monthly averages over the last 10 years.
3. **Seasonal Analysis** — Compare seasonal temperature patterns with statistics for Winter, Spring, Summer, and Autumn.

## Keyboard Shortcuts

| Action                      | Keys                        |
| --------------------------- | --------------------------- |
| Open / Close Help           | `?` or `H`                  |
| Close Help Modal            | `Esc`                       |
| Navigate Decades (Previous) | `←` or `↓`                  |
| Navigate Decades (Next)     | `→` or `↑`                  |
| Jump to Recent Decade       | `1` `2` `3` `4` `5` `6` `7` |

> Number keys jump to recent decades (1960s–2020s). Use arrow keys or the dropdown for earlier periods.

## Features

- **Color-coded temperatures** — Blue for cold, green for mild, yellow/orange for warm, red for hot.
- **Historical data** — Full access to temperature records from 1659 to present across all decades.
- **Seasonal calculations** — Automatic computation of 3-month seasonal averages.
- **Interactive navigation** — Use buttons or keyboard shortcuts to explore data efficiently.
- **Real-time data** — Connects to Met Office database via secure edge function.

## Understanding the Data

**What is CET?**
The Central England Temperature dataset represents mean temperatures calculated from measurements across a triangular area of central England (Lancashire, London, and Bristol).

**Data Quality**
Values marked as "N/A" indicate missing data for that period. The dataset is continuously maintained and updated monthly by the UK Met Office Hadley Centre.
